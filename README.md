This repository contains well-structured Python programs covering Python fundamentals and Object-Oriented Programming (OOP) concepts. It is intended for learning, practice, and revision, progressing from basic concepts to OOP with real-world examples.

**Repository Structure**

02_Data_Types
Basic Python data types and their operations.

03_Data_Types2
Advanced data type operations, indexing, slicing, and methods.

04_Conditionals
Conditional statements using if, elif, and else.

05_Loops
Looping constructs (for, while) and control statements.

06_Functions
Functions, parameters, return values, and function usage.

07_compreh
List, set, and dictionary comprehensions.

08_Generators
Generator functions and the yield keyword.

09_decorators
Decorators and function wrapping concepts.

10_OOP
Object-Oriented Programming concepts including classes, constructors, inheritance, method overriding, and class methods with real-world examples.

Sample
Practice and experimental code snippets.

**Purpose**

Learn Python from basics to OOP

Understand concepts with practical examples

Use as revision notes

Build strong Python fundamentals for advanced domains
